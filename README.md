# Space-Marketing — CRM Integration

Тестове завдання на позицію PHP Developer.

## Про проект

Міні-сайт для роботи з лідами через CRM API. Дві сторінки, форма для відправки ліда і таблиця зі статусами. Всі запити до CRM йдуть через PHP бекенд, токен на фронт не потрапляє.

## Технології

- **PHP OOP** — класи `Config` і `CrmApiClient`
- **Bootstrap 5** — верстка і компоненти
- **REST API** — інтеграція з CRM через cURL
- **Composer** — автозавантаження класів через classmap
- **Apache** — чисті URL через `.htaccess` і mod_rewrite

## Що реалізовано

- Форма з валідацією на фронті (Bootstrap) і на бекенді (PHP)
- Відправка ліда через `addlead` з усіма обов'язковими полями
- IP береться з реального запиту користувача
- Таблиця статусів з фільтром по даті і пагінацією
- Захист `.env` і `src/` від прямого доступу через `.htaccess`

## Локальний запуск

```bash
# 1. Клонуємо
git clone https://github.com/ZalohaD/lead.git
cd lead

# 2. Встановлюємо залежності
composer install

# 3. Створюємо .env
cp .env.example .env
```
